# 第二章 Golang 概述

程序：完成某个功能的指令的集合

Go语言：Google公司创造的语言

## Go语言发展简史

- 2007年原型
- 2009发布
- 2015 Go 1.5
- 2017 Go 1.8
- 2017 Go 1.9
- 2018 Go 1.10

课程使用的版本：1.9.2

## Go语言特点

既能有静态编译语言的安全和性能，又能达到动态语言开发维护的效率

```
Go = C + Python
```

- 指针
- 包：Go语言的每一个文件都要归属于一个包，能不能单独存在
- 垃圾回收
- 天然并发 goroutine
- 管道通信机制
- 函数可以返回多个值
- 切片
