## 前后端分离开发

### 介绍

- 前端开发人员负责前端代码
- 后端开发人员负责后端代码

```bash
Java代码、html、css、js

拆分 =>

后端工程 java          => 打包部署 Tomcat
前端工程 html、css、js  => 打包部署 Nginx
```

### 开发流程

```
定制接口API（请求路径、请求方式、请求参数、响应数据）【关键步骤】

前端开发（mock） <= 并行开发 => 后端开发

联调

提测
```
