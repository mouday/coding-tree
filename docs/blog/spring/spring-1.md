# 一、Spring概述

## spring是什么

Spring是一款主流的Java EE 轻量级开源框架

## Spring的狭义和广义

- 广义的Spring：Spring技术栈
- 狭义的Spring：Spring Framework
    - IOC inverse of control 控制反转
    - AOP Aspect Oriented Programming 面向切面编程


## Spring 特点

- 非侵入式
- 控制反转
- 面向切面编程
- 容器
- 组件化
- 一站式

## Spring 模块组成

- Spring Core
- Spring AOP
- Spring Data
- Spring Web
- Spring Message
- Spring Test
