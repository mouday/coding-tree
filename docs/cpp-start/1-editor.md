## CLion

下载地址

https://www.jetbrains.com.cn/clion/

CLion设置

快捷键

- `Shift + Alt + 键盘上/下` 控制当前行上下移动
- `Ctrl + d` 复制当前行
- `home/end` 移动到行首/行尾
- `Ctrl + 键盘左/右` 光标左右跳单词
- `Ctrl + a` 全选
- `Ctrl + c` 复制
- `Ctrl + v` 粘贴
