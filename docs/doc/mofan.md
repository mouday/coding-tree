# 三阶魔方复原

## 1、三阶魔方结构

颜色：

- 蓝色 - 绿色
- 橘色 - 红色
- 黄色 - 白色

三种块：

| 类型 | 数量(个) | 颜色 |
| -| - | -|
|中心块 | 6 | 1
|棱块 | 12 | 2
|角块  | 8 | 3

## 2、第一步：小白花

目标：

| 0 | 1 | 2 | 3
| - | - | - | -
| 1 |x | 白 | x
| 2 |白 |黄 | 白
| 3 |x  |白 | x

小白花：黄色花心和4个白色花瓣

即：黄色中心块和4个白色棱块

手法：

- 勾：将头顶的白色块移开
- 压：将第3层白色块移动到第2层

步骤：

1. 找到黄色中心块
2. 先拼第二层白色块
3. 把第一、三层的花瓣变到第二层

## 3、第二步：白色十字架

目标：

| 0 | 1 | 2 | 3
| - | - | - | -
| 1 |x | 白 | x
| 2 |白 | 白 | 白
| 3 |x  |白 | x

步骤：

1. 将第一步中黄色面的白色棱块和相同颜色中心块对应
2. 对应后放到右手边，将白色棱块转到白色中心块面
3. 重复4次，将所有白色棱块都转到白色中心块一面

## 4、第三步：白色面和侧面第一层

目标：

白色一面和侧面最底下一层复原

| 0 | 1 | 2 | 3
| - | - | - | -
| 1 |白 | 白 | 白
| 2 |白 | 白 | 白
| 3 |白  |白 | 白

手法：勾上回下

- 右手：左上右下
- 左手：右上左下

步骤：

1. 将白色中心块的那一面朝下
2. 找白色角块，将白色角块另外2个颜色和其中心块对应
3. 将白色角块移动到白色中心块面，如果
    - 白色角块的白色面在侧面，白色角块的白色面朝向自己，如果

        - 白色角块在自己左手边，做左手的`勾上回下`
        - 白色角块在自己右手边，做右手的`勾上回下`
    - 白色角块的白色面朝上，做 `3次` `勾上回下`
        - 白色角块在自己左手边，做左手的`勾上回下`
        - 白色角块在自己右手边，做右手的`勾上回下`
    - 白色角块在第3层（最底层），做 `1次` `勾上回下`，将白色角块移动到上面层，再回到第3步开始处，看具体情况

4. 重复以上步骤4次，将4个白色角块都移动到白色中心块面

## 5、第四步：侧面第二层

目标：

侧面第二层复原

步骤：

1. 将白色中心块的那一面朝下
2. 在黄色中心块面，找到不带黄色的棱块
3. 将棱块颜色和侧面中心块颜色对齐，面对自己
4. 找到和棱块顶部颜色对应的侧面中心块
    - 如果棱块顶部颜色对应的中心块在右边，先用右手做`勾上回下`，白色块面对自己，再做一次左手的`勾上回下`
    - 如果棱块顶部颜色对应的中心块在左边，先用左手做`勾上回下`，白色块面对自己，再做一次右手的`勾上回下`
    - 如果`不带黄色的棱块`在第二层，那么做`2次` `勾上回下`，将棱块移动到顶层
5. 重复上面步骤，直到在黄色面找不到不带黄色的棱块为止，侧面的第二层就复原完成了

## 6、第五步：黄色十字架

目标：

拼出黄色十字架

三种情况：

### 1、一个点

| 0 | 1 | 2 | 3
| - | - | - | -

| 2 |x | 黄 | x
| 3 |x  |x | x

先做 拐角公式 或 一字公式

| 0 | 1 | 2 | 3
| - | - | - | -
| 1 |x | 黄 | x
| 2 |黄 | 黄 | x
| 3 |x  |x | x

拐角公式（右手）：压上右下右上左下提

### 3、一字

| 0 | 1 | 2 | 3

| 1 |x | x | x
| 2 |黄 | 黄 | 黄
| 3 |x  |x | x

一字公式（右手）：上左下右下压上提

## 7、第六步：复原黄色面

目标：

复原整个黄色面

三种情况：

> 注意：三阶魔方不会出现缺1个黄色的情况

### 1、缺2个黄色

| 0 | 1 | 2 | 3
| - | - | - | -
| 1 | x | 黄 | x
| 2 |黄 | 黄 | 黄
| 3 |黄 | 黄 | 黄

先变为小鱼

| 0 | 1 | 2 | 3
| - | - | - | -
| 0 | 黄 | x | x
| 1 | x | 黄 | 黄
| 2 | 黄  |黄 | 黄
| 3 | x | 黄| 黄

左手食指在后面摸到黄色，做左鱼公式

### 2、缺3个黄色

| 0 | 1 | 2 | 3
| - | - | - | -
| 1 | x | 黄 | x
| 2 |黄 | 黄 | 黄
| 3 |x | 黄 | 黄

小鱼

关键：鱼头对着鱼鳃

（1）右小鱼

| 0 | 1 | 2 | 3
| - | - | - | -
| 1 |x | 黄 | 黄
| 2 |黄 |黄 | 黄
| 3 |x  |黄 | x
| 0 |x  |x | 黄

右鱼公式（左手）：下左上左下左左上

（2）左小鱼

| 0 | 1 | 2 | 3
| - | - | - | -
| 1 |黄 | 黄 | x
| 2 |黄 |黄 | 黄
| 3 |x  |黄 | x
| 0 |黄  |x | x

左鱼公式（右手）：下右上右下右右上

### 3、缺4个黄色

| - | 0 | 1 | 2 | 3
| - | - | - | - | -
| 1 | 黄 | x | 黄 | x
| 2 | 2 |黄 | 黄 | 黄
| 3 | 黄 |x | 黄 | x

先变为小鱼

口诀：找眼睛，放左边，做左鱼公式

## 8、第七步：找眼睛

眼睛类型：

| - | 0 | 1 | 2 | 3
| - | - | - | - | -
| 第一种(2+1) | Y | X | Y
| 第二种(3) | X | X | X

目标：变为4双眼睛

情况

1、没有一双眼睛

拐角公式 + 一字公式

1、一双眼睛

眼睛放左边，一字公式 + 拐角公式

## 9、第八步：最后一步

2种情况

情况1、有1双对齐的眼睛 + 3双2+1眼睛

1. 1双对齐的眼睛的颜色对齐中心块，面对自己
2. 找背面眼睛中心的棱块的中心块颜色，是要去左边还是右边
    - 要去右边：先做右鱼公式，再做左鱼
    - 要去左边：先做左鱼公式，再做右鱼

情况2、4双2+1眼睛

做一遍 左鱼公式 && 右鱼公式

## 总结

- 1个手法：勾上回下
- 4个公式：

  - 拐角公式
  - 一字公式
  - 左鱼公式
  - 右鱼公式
