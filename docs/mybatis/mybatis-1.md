# 一、MyBatis简介

## 1、历史

- iBatis = internet abatis
- iBatis 3.x => MyBatis

## 2、特性

MyBatis是一个基于Java的持久层框架

将 Java POJO（Plain Old Java Objects，普通老式 Java 对象）映射成数据库中的记录

MyBatis是一个半自动的ORM （Object Relation Mapping）框架

DAO（Data Access Objects）

## 3、和其他持久化层技术对比

- JDBC 开发效率低
- MyBatis 开发效率尚可
- Hibernate和JPA 开发效率高
